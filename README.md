## esvtoleet

The ESV Bible passage leetspeak translator

#### How to use

```bash
$ python esv-to-leet.py passage key
```

| parameter | description                             |
| :---------| :-------------------------------------- |
| passage   | ex "John 3:16"                          |  
| key       | if you have a key at the ESV Bible Web Service API (http://www.esvapi.org/v2) |

esvtoleet
=========

The ESV Bible passage leetspeak translator

How to use
----------

```
$ python esv-to-leet.py passage key
```

key is if you have a key at the ESV Bible Web Service API (http://www.esvapi.org/v2)
passage is ex "John 3:16"
